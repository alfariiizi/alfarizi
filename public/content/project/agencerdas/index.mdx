## Overview

AgenCerdas is an AI-assisted agent operations platform that integrates WhatsApp workflows with automated support capabilities.

## Business Context

The product helps teams handle customer conversations and agent workflows through a combination of messaging integration, application workflows, and AI-assisted processing.

## My Role

I worked as technical lead, guiding system direction, backend boundaries, implementation quality, and delivery across the team.

## Key Capabilities

- WhatsApp integration
- AI-assisted agent workflow
- Conversation operations support
- Backend service architecture
- Next.js operational interface

## Technical Direction

The system uses Go for backend services, Python for AI-related processing, and Next.js for the application interface. The architecture separates messaging events, AI processing, and operator-facing workflows.

## Outcome / Impact

AgenCerdas demonstrates product delivery across backend architecture, AI-assisted operations, and messaging integration.
