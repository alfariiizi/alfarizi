## Overview

This was my first portfolio website. It was simple, experimental, and built before I had much design experience.

The project was personal, but it became useful professionally because it helped me present my early frontend work.

## My Role

I designed and built the website with React and Tailwind CSS.

## Key Capabilities

- Personal landing page
- Project presentation
- Responsive layout
- Early visual experimentation

This portfolio helped me get my first frontend job at DataIns.
