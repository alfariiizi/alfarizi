## Overview

SPLP Sulawesi Barat is an API gateway management system for supporting data interoperability between government services.

## Business Context

Government services often need structured ways to publish, manage, and monitor APIs. SPLP Sulbar provides a web surface for API publishing, user management, traffic visibility, and tenant information.

## My Role

I contributed as a freelance fullstack developer across the landing page and dashboard surfaces.

## Key Capabilities

- API publishing visibility
- API creation and monitoring information
- Registered user and tenant information
- Traffic summary presentation
- Dashboard workflows for API registration and management

## Technical Direction

The landing page used Next.js and Tailwind CSS. The dashboard used Angular and SCSS. The project required working across different frontend stacks while keeping the user-facing system coherent.

## Outcome / Impact

The project supported API interoperability work for West Sulawesi government services and gave me experience contributing across mixed frontend stacks in a public-sector system.
