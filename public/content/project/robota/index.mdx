## Overview

Robota is an AI-assisted hotel review analytics platform for hospitality businesses. It helps users monitor guest reviews, understand sentiment, and respond to customer feedback more efficiently.

## Business Context

Hotels and hospitality teams deal with review data across online travel agencies. Robota turns that feedback into a more usable operational and analytical surface.

## My Role

I led frontend architecture and implementation work for Robota during my freelance work with DataIns.

## Key Capabilities

- OTA review analytics
- AI-assisted review understanding
- Hospitality dashboard workflows
- Authentication and tenant-aware frontend logic
- Landing page and blog surfaces
- Migration from legacy React to Next.js App Router

## Technical Direction

Robota was migrated from a legacy React architecture into a Next.js App Router application. The stack included Next.js, Tailwind CSS, shadcn/ui, TanStack Query, Zustand, Zod, and tRPC-style type-safe integration patterns.

The migration improved maintainability and gave the product a clearer foundation for server rendering, dashboard workflows, and long-term UI development.

## Outcome / Impact

Robota became one of the main professional projects that shaped my frontend architecture experience. It also led to stronger patterns I later reused in other DataIns and product work.
