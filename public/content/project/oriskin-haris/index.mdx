## Overview

Oriskin HARIS is an internal human resources administration system for Oriskin. The system supports HR workflows used across the company's clinic branch operations.

## Business Context

Oriskin operates more than 80 branches, which creates operational complexity around employee administration, internal records, and cross-branch HR processes.

## My Role

I contributed as a backend engineer, working with Go and supporting the system architecture behind the Next.js interface.

## Key Capabilities

- HR administration workflows
- Branch-aware internal operations support
- Employee data management
- Backend services for internal dashboards
- Structured API boundaries between backend and frontend

## Technical Direction

The backend is built in Go with service boundaries designed for internal operational use. The frontend is implemented with Next.js and consumes backend APIs for administrative workflows.

## Outcome / Impact

HARIS helps centralize HR administration for a company operating a large branch network. Details are intentionally kept high level because the system is internal.
