## Overview

Oriskin NextAppo is an internal appointment management system for Oriskin medical staff. It supports workflows around follow-up and next-appointment planning.

## Business Context

Clinic operations depend on predictable appointment handling. This system helps medical staff manage next-appointment workflows in a more structured internal interface.

## My Role

I contributed as a backend engineer using Go and supported integration with the Next.js interface.

## Key Capabilities

- Next-appointment workflow support
- Medical staff operational views
- Appointment state handling
- Backend APIs for scheduling-related workflows
- Internal dashboard integration

## Technical Direction

The system uses Go for backend services and Next.js for the staff-facing interface. The backend is designed around clear workflow states and internal operational needs.

## Outcome / Impact

NextAppo supports appointment continuity for medical staff workflows. Details are kept high level because the system is internal.
