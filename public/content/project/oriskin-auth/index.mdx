## Overview

Oriskin Auth is an internal authentication and single sign-on system for Oriskin applications.

## Business Context

As internal systems grow, staff need a consistent way to authenticate across applications. A shared auth layer reduces duplicated login flows and improves operational consistency.

## My Role

I worked as a freelance fullstack engineer using Next.js, handling the application layer for internal authentication flows.

## Key Capabilities

- Internal staff authentication
- Single sign-on foundation
- Shared access flow across internal systems
- Secure session-oriented application behavior
- Integration points for other Oriskin applications

## Technical Direction

The project uses Next.js fullstack capabilities to implement auth workflows and application-facing integration points. The focus is on consistency, maintainability, and internal usability.

## Outcome / Impact

The system provides a shared authentication foundation for Oriskin's growing internal application ecosystem.
