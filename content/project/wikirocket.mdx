WikiRocket merupakan wikipedia search engine. Anda dapat mencari banyak topik disini. Semua topik yang anda dapatkan disini merupakan artikel dari wikipedia. Project ini merupakan project pertama sama dengan framework **nextjs** (nextjs 13). Project ini merupakan latihan mandiri saya dari [Dave Gray Nextjs Course](https://youtube.com/playlist?list=PL0Zuz27SZ-6Pk-QJIdGd1tGZEzy9RTgtj&si=loeAEPD9OJof_rb8). Di project ini saya juga pertama kali nya menggunakan typescript pada framework yang berbasis reactjs.

Jika kalian penasaran dengan WikiRocket!, kalian dapat langsung mengunjungi website nya. Link nya sudah saya taruh di bagian atas ya. Semoga project ini bisa menjadi inspirasi untuk project lain anda.
