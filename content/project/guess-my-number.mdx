Ini merupakan game sederhana yang mana user akan menebak angka rahasia antara 1 hingga 20. Jika tebakannya terlalu tinggi, akan ada peringatan "too high" yang berarti angka rahasia nya dibawah itu. Setiap memulai permainan, user diberikan score 20, namun seiring kesalahan jawaban user maka nilai nya akan berkurang -1.

Game sederhana ini merupakan latihan mandiri dari [Course Jonas](https://www.udemy.com/course/the-complete-javascript-course/). Seperti instruksi dari Jonas, game sederhana ini awalnya saya tulis dengan html + css + javascript. Namun ketika saya belajar reactjs, sayapun mencoba untuk membuat ulang game ini dengan reactjs. Dengan bantuan tailwindcss, hasilnya pun menjadi responsive, namun saya baru menyadari bahwa size dari `node_modules` nya cukup besar dikarenakan penggunaan reactjs.

Untuk memainkan game nya, anda membuka link yang sudah saya cantumkan diatas. Hope you enjoy the game!
