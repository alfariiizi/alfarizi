Movientar ini merupakan project pertama saya untuk belajar menggunakan API dari reactjs ke The [Movie DB (TMDB)](https://www.themoviedb.org) API. TMDB merupakan database film dan tv series yang juga menyediakan API dari database nya. Pada project ini, saya menggunakan reactjs dan tailwindcss.

Pada website Movientar ini, ada beberapa Ada beberapa kategori yang dapat dipilih. Jika kalian penasaran dengan website nya, kalian bisa langsung mengunjungi nya. Link nya ada dibagian atas ya. Semoga bisa menjadi referensi di project anda lainnya.
