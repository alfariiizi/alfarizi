Ini merupakan game multiplayer sederhana yang mana user secara bergantian melemparkan dadu. Misalkan ada user-1 dan user-2. Jika user-1 melemparkan dadu dengan nilai 1, maka giliran akan berganti ke user-2. User-1 ataupun user-2 akan tetap dapat melemparkan dadu hingga user-1 atau user-2 mendapatkan nilai 1 atau user menekan button "Hold". Untuk aturan lebih jelasnya, anda dapat membaca [wikipedia](<https://en.wikipedia.org/wiki/Pig_(dice_game)>).

Sama seperti <Link href='/project/guess-my-number'>guess my number</Link>, game sederhana ini juga merupakan latihan mandiri dari [Course Jonas](https://www.udemy.com/course/the-complete-javascript-course/). Seperti instruksi dari Jonas, game sederhana ini awalnya saya tulis dengan html + css + javascript.

Untuk memainkan game nya, anda membuka link yang sudah saya cantumkan diatas. Hope you enjoy the game!
